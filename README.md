# 歯磨き粉の推しメン探し

## サイト概要
皆んなのお気に入りの歯磨き粉に投票してもらい
歯磨き粉のランキングを決めるサイトです

### サイトテーマ
どの歯磨き粉が好きなのか教えて欲しいです
お気に入り１位を決めましょう

### テーマを選んだ理由
前職が歯科衛生士だったこともあり、一般の方々がどのような歯磨き粉を選択して使用してるのか興味があった為

### ターゲットユーザ
20~50代

### 主な利用シーン
どんな歯磨き粉を買えばいいか迷った時参考にして欲しいです
